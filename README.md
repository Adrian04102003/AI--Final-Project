# Live Sign Language to Text and Speech
## Installation
- Clone the repository
- Install python
- Install the requirements
```bash
pip install -r requirements.txt
```
## Running Application
- There was already an existing model, which you can still train it further, too using the non-main.py file
- Run the application using the pyton
```bash
{path}/Sign-Language-to-Text-and-Speech/venv_sls/Scri
pts/python.exe {path}/Sign-Language-to-Text-and-Speech/main.py
```
